# คู่มือใช้งาน QRForPay POS

## เข้าสู่ระบบ
1. เปิดแอพ → กรอกอีเมลและรหัสผ่าน → กด **"เข้าสู่ระบบ"**
2. แอพพาเข้าหน้าหลักทันที

แถบด้านล่างมี 7 แท็บ: **แดชบอร์ด / โต๊ะสด / ออเดอร์ / สินค้า / คลัง / ตั้งค่า / โต๊ะ**

---

## จัดการโต๊ะ (แท็บ "โต๊ะ")

### สีสถานะโต๊ะ

| สี | สถานะ | ความหมาย |
|----|-------|---------|
| 🟢 เขียว | ว่าง | ไม่มีออเดอร์ |
| 🟠 ส้ม | รอทำ | มีออเดอร์รออยู่ ยังไม่ได้ทำ |
| 🟣 ม่วง | กำลังทำ | ครัวกำลังปรุงอาหาร |
| 🔵 น้ำเงิน | พร้อมเสิร์ฟ | อาหารเสร็จ รอเสิร์ฟ/ชำระ |

### ดูรายละเอียดโต๊ะ
- กดที่การ์ดโต๊ะ → เห็นรายการออเดอร์และยอดรวม
- กด **"สร้าง QR"** เพื่อแสดง QR code ให้ลูกค้าสแกนสั่งอาหารเอง

### ยกเลิกรายการอาหาร (Per-Item Cancel)
- กดโต๊ะที่ต้องการ → กด **"ดูออเดอร์"**
- กดปุ่มยกเลิก (X) ข้างรายการที่ต้องการยกเลิก → ยืนยัน
- ยอดรวมจะอัพเดตอัตโนมัติ หากยกเลิกทุกรายการ ออเดอร์จะถูกยกเลิกทั้งหมด

---

## ดูออเดอร์สด (แท็บ "โต๊ะสด")

- เห็นออเดอร์ใหม่ที่ลูกค้าสั่งผ่าน QR ทันที (realtime)
- กด **"เริ่มทำอาหาร"** → สถานะโต๊ะเปลี่ยนเป็น 🟣 กำลังทำ
- กด **"พร้อมเสิร์ฟ"** → สถานะโต๊ะเปลี่ยนเป็น 🔵 พร้อมเสิร์ฟ
- กด **"เสร็จสิ้น / รับเงิน"** → โต๊ะกลับเป็น 🟢 ว่าง

---

## ระบบลูกค้าสั่งเองผ่าน QR

ลูกค้าสแกน QR บนโต๊ะ → เปิดเว็บเมนูบนโทรศัพท์ → สั่งและชำระเงิน PromptPay ได้เอง

**Flow ฝั่งพนักงาน:**
1. ไปแท็บ **"โต๊ะ"** → กดโต๊ะ → กด **"สร้าง QR"**
2. ตั้งจอหรือพิมพ์ QR วางบนโต๊ะ
3. ลูกค้าสั่งเอง → ออเดอร์ขึ้นที่แท็บ **"โต๊ะสด"** ทันที
4. พนักงานดูรายการ → กด "เริ่มทำอาหาร" → ทำ → กด "พร้อมเสิร์ฟ"

**ถ้าลูกค้าจ่ายเงินสดแทน QR:**
- เปิดออเดอร์ที่แท็บ **"ออเดอร์"** → กด **"รับเงิน"** → เลือกเงินสด → ยืนยัน

---

## ดูออเดอร์ย้อนหลัง (แท็บ "ออเดอร์")

- ค้นหาด้วยเลขออเดอร์ได้จากช่องบนสุด
- กรองสถานะ: ทั้งหมด / รอ / กำลังทำ / สำเร็จ / ยกเลิก
- กดการ์ดออเดอร์ → ดูรายละเอียดสินค้าทุกรายการ

---

## รับเงิน QR PromptPay (ฝั่งพนักงาน)

1. เปิดออเดอร์ → กด **"ชำระเงิน"** → เลือก **QR PromptPay**
2. หน้าจอ QR เปิดขึ้น → ส่งมือถือให้ลูกค้าสแกน
3. ลูกค้าสแกนผ่านแอพธนาคาร → โอนเงิน
4. **กดปุ่มสีเขียว "ยืนยันรับเงินแล้ว"** เมื่อเห็นว่าลูกค้าจ่ายแล้ว

---

## รับเงินสด

1. เปิดออเดอร์ → กด **"รับเงิน"** → เลือก **เงินสด**
2. กรอกจำนวนเงินที่รับมา
3. แอพคำนวณเงินทอนให้อัตโนมัติ → กด **"ยืนยัน"**

---

## จัดการสินค้า (แท็บ "สินค้า" — เจ้าของร้านเท่านั้น)

- **เพิ่มสินค้า**: กดปุ่ม **+** มุมขวาล่าง → กรอกชื่อ ราคา สต๊อก หมวดหมู่ รูป → กด **"บันทึก"**
- **แก้ไข**: กดไอคอนดินสอ → แก้ → บันทึก
- **ลบ**: กดไอคอนถังขยะ → ยืนยัน

---

## ตั้งค่าร้าน (แท็บ "ตั้งค่า" — เจ้าของร้านเท่านั้น)

- **ชื่อร้าน / PromptPay ID**: แก้ไขและกด "บันทึก"
- **จำนวนโต๊ะ**: ตั้งจำนวนโต๊ะในร้าน → หน้า "โต๊ะ" จะแสดงตาม grid ที่ตั้งไว้

---

## ปัญหาที่พบบ่อย

**QR สแกนแล้วแต่ระบบไม่ขึ้นสำเร็จ?**
→ กดปุ่มสีเขียว **"ยืนยันรับเงินแล้ว"** ได้เลย ไม่ต้องรอ

**ลูกค้าสแกน QR โต๊ะแต่เมนูไม่ขึ้น?**
→ ตรวจสอบสัญญาณอินเทอร์เน็ตของลูกค้า และตรวจว่า QR ถูก generate จากหน้า "โต๊ะ" ที่ถูกต้อง

**สถานะโต๊ะไม่อัพเดต?**
→ ลากหน้าจอลงจากด้านบน (pull to refresh) หรือออกแล้วกลับเข้าหน้าโต๊ะใหม่

**กดชำระเงินสดไม่ได้?**
→ ต้องกรอกจำนวนเงินที่รับมาก่อน และต้องไม่น้อยกว่ายอดที่ต้องจ่าย

**สินค้ากดไม่ได้?**
→ สต๊อกหมด ให้เจ้าของร้านอัปเดตสต๊อกในแท็บ "สินค้า"

---

## ขั้นตอนสรุป

```
ลูกค้าสั่งผ่าน QR:
สร้าง QR โต๊ะ → ลูกค้าสแกน → สั่งเอง → พนักงานดูที่ "โต๊ะสด" → ทำ → เสิร์ฟ ✅

พนักงานขายเอง:
โต๊ะสด → เพิ่มสินค้า → ตรวจตะกร้า → ชำระเงิน → ยืนยัน → เสร็จ ✅
```
