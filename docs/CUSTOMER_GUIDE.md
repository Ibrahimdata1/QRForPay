# คู่มือใช้งาน EasyShop POS

## เข้าสู่ระบบ
1. เปิดแอพ → กรอกอีเมลและรหัสผ่าน → กด **"เข้าสู่ระบบ"**
2. แอพพาเข้าหน้าขายทันที

แถบด้านล่างมี 5 แท็บ: **แดชบอร์ด / ขาย / ออเดอร์ / สินค้า / ตั้งค่า**

---

## ขายของ

### 1 — เลือกสินค้า (แท็บ "ขาย")
- กดการ์ดสินค้าเพื่อเพิ่มลงตะกร้า
- กรองหมวดหมู่ได้จากปุ่มแถบบน
- สินค้าสีแดง = สต๊อกหมด กดไม่ได้

### 2 — ตรวจตะกร้า
กดปุ่ม **"ตะกร้า"** ที่ลอยอยู่ด้านล่าง
- กด + / − เพิ่มลดจำนวน
- กดแถว **"เพิ่มส่วนลด"** เพื่อลดราคา (% หรือ ฿)
- ยอดล่างสุด = ยอดที่ลูกค้าจ่ายจริง (รวม VAT แล้ว)

### 3 — ชำระเงิน
เลือกวิธีจ่าย แล้วกด **"ชำระเงิน ฿XXX"**

---

## รับเงิน QR PromptPay

1. เลือก **QR PromptPay** → กด **"ชำระเงิน"** → กด **"ยืนยัน"**
2. หน้าจอ QR เปิดขึ้น → ส่งมือถือให้ลูกค้าสแกน
3. ลูกค้าสแกนผ่านแอพธนาคาร → โอนเงิน
4. **กดปุ่มสีเขียว "ยืนยันรับเงินแล้ว"** เมื่อเห็นว่าลูกค้าจ่ายแล้ว
5. ออเดอร์บันทึกสำเร็จ ตะกร้าล้างอัตโนมัติ

> ถ้าลูกค้ายังไม่จ่ายหลัง 5 นาที → กด **"ยกเลิก"** แล้วเริ่มใหม่

---

## รับเงินสด

1. เลือก **เงินสด** → กรอกจำนวนเงินที่รับมา
2. แอพคำนวณเงินทอนให้อัตโนมัติ
3. กด **"ชำระเงิน"** → ยืนยัน → เสร็จ

---

## ดูออเดอร์ย้อนหลัง (แท็บ "ออเดอร์")

- ค้นหาด้วยเลขออเดอร์ได้จากช่องบนสุด
- กรองสถานะ: ทั้งหมด / รอ / ยืนยันแล้ว / สำเร็จ / ยกเลิก
- กดการ์ดออเดอร์ → ดูรายละเอียดสินค้าทุกรายการ

---

## จัดการสินค้า (แท็บ "สินค้า" — เจ้าของร้านเท่านั้น)

- **เพิ่มสินค้า**: กดปุ่ม **+** มุมขวาล่าง → กรอกชื่อ ราคา สต๊อก หมวดหมู่ รูป → กด **"บันทึก"**
- **แก้ไข**: กดไอคอนดินสอ → แก้ → บันทึก
- **ลบ**: กดไอคอนถังขยะ → ยืนยัน

---

## ปัญหาที่พบบ่อย

**QR สแกนแล้วแต่ระบบไม่ขึ้นสำเร็จ?**
→ กดปุ่มสีเขียว **"ยืนยันรับเงินแล้ว"** ได้เลย ไม่ต้องรอ

**QR หมดเวลา?**
→ กด "ยกเลิก" → กลับตะกร้า → กดชำระใหม่

**กดชำระเงินสดไม่ได้?**
→ ต้องกรอกจำนวนเงินที่รับมาก่อน และต้องไม่น้อยกว่ายอดที่ต้องจ่าย

**สินค้ากดไม่ได้?**
→ สต๊อกหมด ให้เจ้าของร้านอัปเดตสต๊อกในแท็บ "สินค้า"

**ข้อมูลไม่อัปเดต?**
→ ลากหน้าจอลงจากด้านบน (pull to refresh)

---

## ขั้นตอนสรุป

```
ล็อกอิน → เลือกสินค้า → ตรวจตะกร้า → ชำระเงิน → ยืนยัน → เสร็จ ✅
```
